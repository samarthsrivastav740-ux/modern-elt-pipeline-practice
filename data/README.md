# Dataset

## Source

This project uses the **MovieLens 20M Dataset**, a publicly available movie recommendation dataset maintained by GroupLens Research at the University of Minnesota.


## Files Used

| File | Description |
|------|-------------|
| movies.csv | Movie information including title and genres |
| ratings.csv | User ratings for movies |
| tags.csv | User-generated tags for movies |
| links.csv | Mapping to external movie database IDs |
| genome-tags.csv | Tag definitions |
| genome-scores.csv | Relevance scores between movies and genome tags |

## Why this dataset?

The MovieLens dataset provides multiple related tables, making it suitable for practicing:

- Cloud data ingestion
- Data warehousing
- SQL transformations
- Dimensional modeling
- Building fact and dimension tables
